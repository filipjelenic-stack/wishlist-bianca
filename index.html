Настройка общих броней

Чтобы все гости видели одни и те же брони, нужно один раз создать бесплатное хранилище на jsonbin.io и вставить два ключа в файл.

  1. Зайдите на jsonbin.io → Sign Up (бесплатно)
  2. Нажмите + New Bin, вставьте {}, нажмите Create
  3. Скопируйте BIN ID — длинный код в адресной строке после /b/
  4. В меню API Keys создайте ключ и скопируйте его
  5. Откройте index.html в любом текстовом редакторе (Блокнот, VS Code)
  6. Найдите строки в начале скрипта:
    const BIN_ID = "";
    const API_KEY = "";
    и вставьте свои значения внутрь кавычек
  7. Выложите файл на GitHub Pages (бесплатно) — и готовая ссылка делится с гостями!

💡 Пока ключи не заполнены — брони сохраняются только в вашем браузере (localStorage). Всё остальное работает полностью.
